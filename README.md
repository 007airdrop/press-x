# Press To Start - Farcaster Frame v2

## Setup
1. `npm install`
2. Update `.env.local` with your CDP Key and URL.
3. Deploy to Vercel.
4. Test in Warpcast Frame Debugger.

## Features
- Rarity Engine: Common(50), Uncommon(30), Rare(15), Legendary(5)
- Glassmorphism UI
- Coinbase Smart Wallet Integration
- Framer Motion Transitions
